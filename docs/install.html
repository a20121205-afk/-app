iPad 使用教學

把家訪紀錄工具加入主畫面

這個工具是 PWA 網頁 App，在 iPad 上不需要從 App Store 安裝，只要用 Safari 開啟後加入主畫面，就會更像真正的 App。

回到家訪紀錄工具

三步完成安裝

請務必使用 iPad 的 Safari 開啟工具網址，再依序完成下面步驟。

1

用 Safari 打開工具網址

請先在 iPad 上用 Safari 開啟這個工具，不要從 LINE、Messenger 或其他 App 內建瀏覽器打開。

重點： 只有 Safari 比較穩定支援「加入主畫面」與相機權限。
2

點分享按鈕

在 Safari 底部或頂部工具列找到分享按鈕，圖示通常是方框上方有一支向上的箭頭。

如果看不到： 先輕點頁面一次，讓 Safari 工具列重新出現。
3

選加入主畫面

在分享清單往下找到「加入主畫面」，確認名稱後按「加入」，之後桌面就會出現這個工具圖示。

完成後： 之後從主畫面直接開啟，體驗會比一般瀏覽器分頁更像 App。

使用前提醒

這些細節先確認好，iPad 使用會更穩。

需要 HTTPS 網址

如果你要使用拍照、加入主畫面、離線快取等功能，建議把工具部署在 HTTPS 網址上，iPad 支援會比較完整。

草稿先存在這台 iPad

目前草稿會先存在這台 iPad 的瀏覽器資料庫裡，所以如果換裝置或清除 Safari 網站資料，草稿不會自動同步過去。

如果看不到加入主畫面

常見原因是現在不是用 Safari 開啟，或網址還不是正式網站。可以先改用 Safari，或重新確認部署網址是否正常。

建議做法

如果你要固定給 iPad 使用，下一步最適合把這份 PWA 放到 GitHub Pages 或其他 HTTPS 網址，這樣安裝最方便。
