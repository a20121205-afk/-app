# 個管家訪紀錄輔助工具

這是一個可本機開發、可正式打包、也可部署到靜態主機的前端專案，重點功能包含：

- 填寫家訪基本資料
- 勾選 ADL / IADL 功能狀況
- 記錄主要照顧者與次要照顧者
- 輸入家訪觀察與家屬主述
- 直接開啟相機拍照或上傳照片
- 將資料先存成草稿到本機瀏覽器資料庫
- 自動整理出家訪紀錄、功能狀況描述、問題清單、建議方向

## 專案結構

- `index.html`：頁面入口
- `app.js`：主要互動邏輯
- `styles.css`：畫面樣式
- `public/`：網站圖示與 manifest
- `scripts/`：目前這台電腦可直接用的 portable Node 啟動腳本

## 標準指令

- `npm run dev`：啟動開發模式
- `npm run build`：打包正式版
- `npm run preview`：預覽打包結果

## 目前這台電腦可直接執行

如果系統沒有全域安裝 Node，可直接用這幾個 PowerShell 腳本：

- `.\scripts\run-dev.ps1`
- `.\scripts\run-build.ps1`
- `.\scripts\run-preview.ps1`

## 打包輸出

- 正式部署檔會在 `dist/`
- 這是靜態網站輸出，可部署到 Vercel、Netlify、GitHub Pages 或一般虛擬主機

## iPad / PWA 使用方式

- 建議先把 `dist/` 部署到 HTTPS 網址
- 在 iPad 上用 Safari 開啟
- 點分享，再選「加入主畫面」
- 第一次開過後，常用靜態檔會被快取，離線時仍可開啟已載入過的版本
- 若要使用直接相機功能，建議使用 HTTPS 網址開啟

## 目前狀態

- 目前是前端本地整理版
- 照片與草稿先存放在目前裝置的瀏覽器中，不會自動上傳雲端
- 文字整理邏輯是依規則輸出，還沒有串接真正的 AI API
